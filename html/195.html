IP获取和定位

搜索
系统内置了get_client_ip方法用于获取客户端的IP地址，使用示例：$ip = get_client_ip();如果要支持IP定位功能，需要使用扩展类库ORG.Net.IpLocation，并且要配合IP地址库文件一起使用，例如：import('ORG.Net.IpLocation');// 导入IpLocation类
$Ip = new IpLocation('UTFWry.dat'); // 实例化类 参数表示IP地址库文件
$area = $Ip->getlocation('203.34.5.66'); // 获取某个IP地址所在的位置如果传入的参数为空，则会自动获取当前的客户端IP地址，要正确输出位置，必须配合UTF8编码的ip地址库文件，否则可能还需要进行编码转换。IP地址库文件和IpLocation类库位于同一目录即可。
